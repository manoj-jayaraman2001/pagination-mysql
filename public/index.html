Pagination API using MySql, Node and Express

I have developed a pagination API using MySQL. It includes optional parameters for limit and offset, with default values of 10 and 0 respectively. If the parameters are not specified or are invalid, the API seamlessly falls back to the default values, ensuring reliable pagination behavior.

Note: Maximum orders are 40, Incase of invalid values for offset and limit such as string, float, negative values or value that exceeds total values, default vales are taken(limit = 10 and offset = 0)
